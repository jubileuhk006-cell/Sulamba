# Dragon Ball Z Fighting Game

## Overview
This project is an exciting Dragon Ball Z themed fighting game developed using the Godot Engine. Players will embody their favorite characters from the series, engaging in intense battles with both AI and other players.

## Features
- **Multiple Characters:** Play as various iconic Dragon Ball Z characters, each with unique abilities and moves.
- **Stunning Graphics:** Utilize Godotâ€™s capabilities to create vibrant and detailed environments based on Dragon Ball Z settings.
- **Fluid Combat Mechanics:** Experience dynamic combat with combos, special moves, and transformations.
- **Local and Online Multiplayer:** Fight against friends locally or challenge players online.
- **Story Mode:** Follow a narrative that incorporates key events from the Dragon Ball Z storyline.

## Getting Started
To get started with the project, clone the repository and open it in Godot Engine. Ensure you have the latest version of Godot installed.

1. Clone the repository:

3. Follow the README for further instructions on setting up assets and gameplay features.
